Data Binding

Interpolation

Syntax: {{'{{'}} Data {{'}}'}} Description: It's the simplest form of data binding. With interpolation, you can incorporate string values directly from your component into your HTML templates.

{{ data_binding_value }}

Property Binding

Syntax: [property]="data" Description: It allows you to bind a DOM property to a component's property. It's unidirectional, meaning data flows from the component to the view. The Image below is showed using "Property binding", from the logic class (.ts) to the element (.html).

Event Binding

Syntax: (event)="handler()" Description: It allows you to respond to user actions (like clicks, mouse movements, key presses, etc.) by calling a method in the component. Data flows from the view to the component.

Two Way Data Binding

Syntax: [(ngModel)]="data" Description: It's a combination of property binding and event binding. It allows for data to flow in both directions: from the view to the component and vice-versa. It's often used with form elements.
